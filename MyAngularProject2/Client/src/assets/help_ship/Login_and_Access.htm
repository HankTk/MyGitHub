Accessing SMARTShip

To access SMARTShip:

  1. Open your web browser and in the address bar, enter the following URL:

https://smartship.alphaorimarine.com/

The SMARTShip login window appears:

The recommended web browser for using SMARTShip application is Google Chrome

  1. Complete the following the details:

  • User Id - Enter your SMARTShip user ID

  • Password - Enter your SMARTShip password

  1. Click LOGIN. The SMARTShip Vessel View appears:

Please note that if the Shore SMARTShip application is upgraded to the latest version and the ship is not, your previously downloaded reports and RMD favorites from the previous version becomes unavailable. You have to recreate them.
