SMARTShip - Overview

SMARTShip as A DIGITAL ENTERPRISE is an Industrial Internet of Things (IIoT) based system capable of serving multiple  vessels of your company. It connects all the components or equipment on-board the vessel with each other as well as to the Internet. This makes the data on every single component available to the SMARTShip user which in turn allows remote monitoring, diagnostics and control, decision support systems, real-time analytics, maintenance and so on.

How it Works?

The SMARTShip system collects data from the various assets or components of your vessel using data acquisition units such as sensors or actuator modules installed on-board the vessel. This acquired data is used by the SMARTShip application which acts as a central console to locally or remotely monitor and manage the connected devices. The data acquired from the data acquisition units can also be used for analysis and maintenance for increasing the over all operational efficiency and cost optimization of your vessel.

Features and Benefits

The following features of SMARTShip can be used for the smooth operation of your vessel:

  • A Dashboard which provides a detailed outlook of the vessels that are assigned to you. It also provides the various information associated with the vessels such as Alerts, Reports, Cases and so on.

  • Alert Management System for generating alerts based on the data acquired from the data acquisition units and the conditions that you provide. Alerts can be subscribed via Dashboard, Email or SMS. Example: You may want to generate an alert when the temperature of a specific asset is beyond the recommended value.

  • Reports based on pre-defined templates for analysis and better decision making.

  • Remote Monitoring and Diagnosis (RMD) display live data fetched from sensor in a graphical format

  • Live streaming of assets for monitoring.

  • Case Management system to centralize, manage and process information about a complex problem or issue.

  • Role Management which allows you to create roles with access to specific features of your SMARTShip system.

  • User Management for creating and managing users. You can also assign roles to users from here.

Using the above capabilities of SMARTShip, you are able to achieve the following benefits:

  • Cost optimization by real-time monitoring of devices and on-board machinery using a centralized system

  • Increase the overall operational efficiency by proactive and corrective actions using real time data

  • Data security
