Vessel Report

This report captures the general details of the vessel along with other data such as cargo and stability, speed, distance, weather, engine details and so on.

To access Vessel Report:

  1. From the Apps area in Vessel View, select Reports and select Vessel Report. The  Vessel Report pane appears on the right:

This pane shows reports which have been previously generated. You can also search for the previous instances in a particular time period using Search.

  1. To view an instance of a report, click View under Actions. The report instance appears:

  1. To download the instance of the report in PDF, click Download under Actions.

You can download or print the report from the report instance. See the below image for reference

If you wish to go back to the Report listing area from the report instance, click on the top-right corner of the Report instance. See the below image for reference.

The View and Download options under Actions is for reports that are generated based on the frequency specified while creating a report configuration. If you wish to generate the latest instance of the report:
