Alert Management

The Alert Management area allows you to create, edit and view alerts.

To access Alert Management:

  • Click Settings and select Alert Management from the menu. The Alert Management window appears:

Various Fields of Alert Management Pane

You can view the alerts from the Alert Management page using the following two filters:

  • Management Alert - This is selected by default. Displays all the alerts which are under your company.

  • Vessel Alert - Select to view alerts belonging to a specific vessel. Upon selection, you are able to further select the vessel from a drop-down menu.

You can also search for vessel specific alerts using the Search field

The display pane shows the list of alerts that are created. It contains the following fields:

  • Name - This is the name of the alert. The category of the alert also appears along with this.

  • System Name - This is the category of system to which the alert belongs. The following are the four types of systems:

    • Decision support system

    • Engine and machinery

    • Hull and cargo

    • Navigational system

  • Alert Type - Indicates the type of alert. It can belong to the following category:

    • Technical

    • Operational

    • Compliance

    • Others

  • Description - Displays the description for the alert.

  • Created Date - Displays the date when the alert was created.

  • Created By - Displays the user who created the alert
