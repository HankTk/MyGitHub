My Profile

Once you log on to your SMARTShip application, you would want to check your profile details and make any changes, if required. This feature allows you to edit your profile details on your SMARTShip system. It also allows you to logout from the application.

To access My Profile:

  • From the SMARTShip dashboard, hover over to the top-right corner and click My Profile . The following menu appears:

The following details appear here:

  • Profile Name - Displays the name of the logged in user.

  • Profile - Allows you to change the details of your profile. For more information on this, refer to Change Profile Settings.

  • Logout - Allows you to logout from the SMARTShip system.
