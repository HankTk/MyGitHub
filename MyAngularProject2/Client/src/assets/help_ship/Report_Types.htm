Report Types

The reports configured using Report Configuration are available under Apps in the vessel view. The following reports are available under Apps:

  • Deck & Engine Report

  • DG Performance Report

  • ECA/SECA Report

  • Vessel Report

  • Voyage Plans

  • TFOC 3 Hourly Report

The number against the Reports icon indicates the number of notifications for Reports.

If you want to create or subscribe to new report configurations or edit an existing one, refer to Report Configuration
