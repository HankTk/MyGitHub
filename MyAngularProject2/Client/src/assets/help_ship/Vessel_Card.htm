Vessel Card

The vessel card appears below the Home icon. It displays the details of the vessel.

The following details appears in the vessel card:

  • Vessel Name

  • IMO Number

  • Vessel Type

  • Actual Time of Departure (ATD) of the journey

  • Estimated Time of Arrival (ETA) of the journey

  • Starting Location of the journey

  • Destination of the journey

The green arrow at the bottom of the card indicates the journey status. The following are the two types of statuses.

  • At Anchor - If the status is 'At Anchor', the green arrow at the bottom of the vessel card is placed till the end of the line.

  • Underway using Engine - If the status is 'Underway using Engine', the green arrow is placed at a point in the line based on the number of days the vessel is in transit. This indicates the distance covered.
