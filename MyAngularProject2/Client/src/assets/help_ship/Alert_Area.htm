Alert Pane

An alert is a notification generated when a certain condition is met. In SMARTShip, an alert is generated when an input condition or combination of conditions  is met by a sensor or an asset.  

Example: You may want to generate an alert when the value from the sensor 'BOOSTER_PUMP_DISCHARGE_PRESSURE_HIGH' is greater than 100.

The alert bar appears on the left-hand side of your SMARTShip Vessel View. All your existing as well as subscribed alerts appear here.

The colored alert bar in the alert pane categorizes alerts based on their severity. Clicking on the required color category displays the alerts corresponding to it. Only active alert count appears here.

The list view has the list of all alerts in a page format which you navigate. The list view has the following fields:

  • Alert Name - Name of the alert

  • System - System to which the alert belongs

  • Alert Type - Type of the alert (operational, technical and so on)

  • Alert Description - Description of the alert

  • Active/Inactive - Indicates whether the alert is in active or inactive status

  • Last Updated - Indicates the last updated time of the alert

If you wish to create or subscribe to new alerts or edit an existing one, refer to the topic Alert Management
