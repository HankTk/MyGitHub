Create Alert

To create a new alert:

  1. From the Alert Management window, click Create New. The Alert Management window appears:

The left pane allows you to choose whether you want to create a management alert or a vessel specific alert. The right pane allows you to select the sensors and the set the conditions for the sensors. These conditions when satisfied triggers the alert.

Create a Management Alert

  1. Complete the following fields:

  • Alert Name - Enter a name for the alert.

  • Description - Enter a brief description for the alert.

  • Systems - Select the required system category from the list.

  • Time delay in seconds - Enter or select the time delay in seconds. An alert is generated only after the alert condition remains satisfied until the time period specified here.

  • Alert Type - Select the required alert type category from here.

  • Alert Priority - Select the priority of the alert from here. The default alert priority is 'Medium'.

  • Probable causes and suggested measures - Enter the probable cause and the suggested measures for the alert condition, if there are any.

  • Subscribers - Allows you to assign the alert to specific role types. It also allows you to enforce the alert and make it mandatory to acknowledge.

  1. Make the required selections and click Assign Role. Once this is completed, follow the steps in Add Alert Conditions.
