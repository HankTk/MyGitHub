Apps in Vessel View

The Apps area from the vessel view allows you to access various assets, reports and cases for the corresponding vessel. The Apps area appears at the bottom of your vessel view.

If there are numbers against the icon, then it indicates the number of active notifications from the respective app.

In the above image, the number '8' indicates that there are 8active notifications from the Reports app.

Hover over the required App icon and the app name appears. The various apps are detailed below:

  • TFOC - Total Fuel Oil Consumption (TFOC) is a next generation voyage optimization application which is integrated into the SMARTShip system. It provides the total fuel and oil consumption for your vessel's voyages. It also uses various methods to provide voyage route recommendations based on the collected data.

  • Reports - Allows you to access the various subscribed report configurations. The different types of reports you can access from here are:

    • Deck & Engine Report

    • DG Performance Report

    • ECA/SECA Summary Report

    • Vessel Report

    • Voyage Plans

    • TFOC 3 Hourly Report

 For more information on this, refer to Reports Overview and Report Types

  • RMD - Real-time Monitoring and Diagnosis (RMD) allows you to display live data fetched from sensors installed on-board the vessels. For detailed information on this, refer to RMD Overview.

  • Chat - Allows you to chat between the various users of your SMARTShip system for any communication purposes. Click and the chat area appears below:

The chat conversations are only between the SMARTShip users of the selected vessel

  • Cases - Allows you to create and edit cases. For detailed information on this, refer to Case Management Overview.
