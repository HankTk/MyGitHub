Create a Report

To create a new report:

  1.  From the Report Configuration List window, click Create New. The Report Configuration window appears:

  1. Complete the following fields:

  • Report Template - Select the required report template type from the list.

For detailed information on the various report templates, refer to Report Types.

  • Vessel Name - Select the required vessel to associate from the list.

  • Frequency - Select the required frequency at which the report is to be generated. You can select either of the following:

    • Daily - Select, if you want to generate the report on a daily basis. If selected, you can specify the time to generate the report in the Time field. See the below image for reference.

You can also manually enter the required time to generate the report in the Time in UTC field.

The time generated is in UTC and not in your local time

    • Weekly - Select, if you want to generate the reports on a weekly basis. If selected, you can specify the days and the time to generate in the Days and Time fields respectively.

  • Export As - Select the format in which the report is generated. You can select either of the following:

    • PDF

    • HTML

  • Send to - Select the required users to send the report from the list.

  • CC - Select the users whom you want to copy the report when generated. You can also manually enter any valid Email address in this field to copy the report.

  1. Click Save. The report is created and appears in the Report Configuration list.
