Accessing RMD

To access and view the graphical representation of sensor data:

  1. From the RMD area, hover over the icon which changes to Create. Select this and the sensor listing pop-up appears:

As an example, the Navigation category is selected and the list of subcategories appears below it.

  1. Select the required system category and sub system names to which the sensor belongs appear:

As an example. ANEMOMETER is selected and the WIND SPEED AND ANGLE subsystem which comes under it is selected.

  1. Select the required sensors and click Show Graph. The streaming occurs and the graph is plotted with the data from the selected sensors.
