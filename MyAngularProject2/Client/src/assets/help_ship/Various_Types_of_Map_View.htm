Various Types of Map View

There are three different types of views for maps from the vessel view. They are:

  • Weather - On selection, the map view gets overlaid with the weather view which indicates the weather in the global map. The vessel marker remains in the weather view.

You can select the following weather properties for further details from the weather map view:

    • Wind

    • Temp

    • Pressure

    • Swellperiod

    • Current

    • Waves
