abs in Alert Area

The alerts area has the following three tabs:

  • Active - This is the default tab. An alert appears here until the alerts conditions returns to normal. The read and unread  icons indicates whether the alert is read or not. Once the alert conditions returns to normal, the alert appears in the All tab if read by the user. If not, it appears in the Unread tab as well the All tab.

  • Unread - All the unread alerts appear here. Once an alert is read, it is removed from this tab.

  • All - All alerts which are active, inactive, read or unread appear here.
