Actions from RMD

You can do the following actions from here to customize the graph data accordingly:

  • View History - Allows you to plot the graph using the historic data from sensors by selecting the required date from the date selection box. You can also specify the required time as well.

  • Live Data - Allows you to plot the graph using the current data from selected sensors.

  • Display by Time Interval - This option allows you to display the graph data in selected time intervals. You can view the graph data in a time interval of 15 minutes, 30 minutes and 1 hour. The graph view changes to the data from the sensors in the last 15 minutes, 30 minutes or one hour based on your selection. See the below images for reference.

  • Group  - Allows you to group and all the sensors together and plot the graph lines accordingly.

  • Legend - Allows you enable and disable the sensor pane on the right. Disabling the pane expands the chart to the right.

  • Add More Sensors - Allows you to add more sensors to your current selection of sensors. You are directed to the Sensor Listing area. Select the required additional sensors and click Show Graph. The additional sensors are added to the RMD graph area.

  • Favorites - Allows you to add the current selection of sensors and the diagnostic chart as a favorite to the My Favorites tab in Ship Assets View as well as the favorite charts in RMD. Click and the Save Favorite box appears:
