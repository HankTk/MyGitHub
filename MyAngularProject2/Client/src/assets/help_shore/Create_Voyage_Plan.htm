Create Voyage Plan

To create a voyage plan for TFOC:

  1. From the ship version of the SMARTShip system, go to the vessel view of the corresponding vessel.
  2. From the Apps section below and select Reports. The Reports area appears:


  1. Select Voyage Plans. The Voyage Plans pane appears on the right:

The Voyage Name pane lists the voyages that are already created. The Actions pane lists the actions that are available for each existing voyage.

The search field above allows you to search voyages in a specific time period.

  1. Click New Voyage Plan to create a new voyage. The New TFOC Voyage Plan window appears:

The vessel name and IMO number is automatically selected based on the vessel.

  1. Complete the remaining fields:

  2. Start Port Name

  3. Start Port Latitude  

  4. Start Port Longitude  

  5. Destination Port Name

  6. Destination Port Latitude  

  7. Destination Port Longitude  

  8. Actual Time Departure (ATD)

  9. Earliest Estimated Time of Arrival (ETA)

  10. Just In Time Estimated Time of Arrival (ETA)

  11. Displacement (in MT)  

  12. Fuel Density  

  13. Charter Party Fuel Consumption (in MT)

  14. Charter Party Speed (in KN)

The latitude and longitude values are validated

  1. Once the details are entered, click Save. The Voyage is created with the message highlighted in the image below:

Here, a voyage for Dublin-Capetown is created.

Once the voyage is created, it may take up to 30 minutes in order to reflect in the Voyage Route list of TFOC. This is based on the computing time required by the TFOC app to calculate the predictions as well your network connectivity.
