Various Tabs of an Asset System

A system is a group of sensors associated to a particular asset or equipment. It provides all the data incoming from the associated sensors in the form of graphs, bars, and data gauges. It also provides a list view of all the sensors and the complete details of the data associated with it.

To access a system:

  1. From the required asset category tab from Ship Assets View, select the required system. The system area appears:

This area has the following four tabs:

  • Overview

  • List View

  • Diagnostics

  • Documents

Please note that the various forms of data representation such as data gauges, bar graphs, tables and so on are available only if the data tags corresponding to it is available as well.
