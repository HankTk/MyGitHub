Features of TFOC

The TFOC app consists of the Voyage Route drop-down menu which displays the historic, current and future voyages for the corresponding vessel. When the app opens, the current or the active voyage appears as shown in the image below. An active voyage is denoted as 'IN PROGRESS'.

If there no active voyages, the message 'No Active Voyage Found' appears. Select the required historic or future voyage from the Select Voyage drop-down menu

Historic voyages are denoted as 'COMPLETED' and future voyages are denoted as 'PLANNED'.

The Create/Edit Voyage feature is available only in the ship version of the SMARTShip application and is disabled in the shore version.

Below the voyage selection list, the following fields appear:

  • Total Voyage Savings - Displays the actual savings in dollars calculated for the route followed by the vessel. For active voyages, the value reflects the savings as of last 3-hourly report and shows the total value once the voyage is completed. It shows zero for voyages which have not yet started.

  • Total CO2 Savings - Shows the carbon emission savings in metric ton. For active voyages, the value reflects the savings as of last 3-hourly report and shows the total value once the voyage is completed. It shows zero for voyages which have not yet started.

  • Total Fuel Savings - Shows the fuel savings in metric ton. For active voyages, the value reflects the savings as of last 3-hourly report and shows the total value once the voyage is completed. It shows zero for voyages which have not yet started.

The voyage information bar appears below this. It displays the following voyage details:

  • IMO Number

  • Displacement

  • Start Port

  • Destination Port

  • Earliest ETA

  • Just in Time ETA

The voyage is represented using three different line graphs in the voyage map area. They are:

  • Predicted - This is based on the Initial Total Fuel Consumption prediction along with detailed voyage path, schedule and speed at the start of the Voyage. It considers the optimum sailing path, weather forecasts, vessel displacement, vessel type and other vessel metadata.

  • Forecast - This route is provided by TFOC. It is the optimized forecast fuel prediction (based on best weather or best fuel efficient) from the current location to the end of the voyage.

The forecast line is taken off from the map area once the voyage is completed.

  • Reported - This is the actual route taken by the ship for the voyage (taken from the vessel report).

The Advice to Master ribbon above the voyage map shows the advisory data for an active journey. It suggests the next location coordinates to head to as well as the speed with which your vessel should move. This is in order to achieve the forecasted route suggested by TFOC.

  • If you wish to view the advice for all the days of the voyage, click See Advice From All Days. The Voyage Plan Data table appears:

The advisory data here can be filtered using two methods:

  • Plan Type

  • Granularity

Plan Type

The Plan Type filter allows you to view the advisory data for fuel efficiency, best weather and a hybrid of both. It also provides the advisory data for reaching your destination using the fastest recommended route. There are multiple variations for fuel efficiency, best weather and the hybrid combinations.

 

The descriptions of these plan types are given below:

  • Most Fuel Efficient 1 and 2 - Provides the top two detailed voyage plans which includes speed, weather, bearing and waypoints from the current location to the end of the voyage having the lowest total fuel consumption.

  • Best Weather 1 and 2 - Provides the top two detailed voyage plans which includes speed, weather, bearing and waypoints from the current location to the end of the voyage considering the best weather along the way.

  • Hybrid 1 and 2 - Provides two detailed voyage plans including speed, weather, bearing and waypoints from the current location to the end of the voyage while optimized between the lowest fuel consumption and the best weather. In most circumstances where the weather is not too adverse along the route matches the best fuel efficient plan.

  • Fastest - Provide the detailed Voyage plan including speed, weather, bearing and waypoints from the current location to the end of the voyage to reach the destination at the shortest possible time.

Granularity

Allows you to select the voyage plan data for the selected plan type for every 6 hours, 12 hours, 24 hours and for all.

In the Voyage Route tab, the ribbon below shows the following daily fuel consumption and savings on a daily basis:

  • Daily Savings

  • CO2 Savings

  • Fuel Savings

  • Fuel Used

  • Distance Travelled

The Total Fuel Consumption area below shows the total fuel consumption for the entire voyage in a graphical format. It showcases the predicted, reported, best case and worst case numbers for the fuel consumption. It also provides a future forecast fuel prediction for the rest of voyage from the last received location. The lines are plotted based on the data from the TFOC 3 Hourly Reports.  

The fuel graphs appearing in this area is explained below:

  • Predicted - This is the initial total fuel consumption prediction along with detailed voyage path, schedule and speed at the start of the voyage.  It is calculated considering the optimum sailing path, weather forecasts, vessel displacement, vessel type and other vessel metadata.

  • Reported - This is the total fuel consumed by the main engine in tonnes as reported from the vessel either by Captain’s Noon Report or SMARTShip Voyage Report.

  • Forecast - This is the optimized forecast fuel prediction from the current location to the end of the Voyage. This line is taken off once the voyage gets completed.

  • Best Case - This is the calculated optimized total fuel consumption between the last two reported locations considering the optimized sailing path, actual weather along the way points, and other vessel parameters such as displacement, vessel type, possible speeds and so on.

  • Worst Case - This is the total fuel consumption based on captain's reported weather data.. Here, instead of considering the actual weather along the sailing path, TFOC uses the captain's reported weather data and shows the worst possible fuel consumption for the voyage. 

The Spreadsheet feature on selection displays the predicted, reported, best case and worst case fuel consumptions for each predicted as well as reported location coordinates for the entire voyage.

To view 3 hourly reports, select View / Edit 3 Hourly Reports. For more information on these reports, refer to TFOC 3 Hourly Report


 

 When the forecast calculations are being processed, the forecast data in map, advice for all days, chart & spreadsheet are not displayed if the data is not received from TFOC backend.
