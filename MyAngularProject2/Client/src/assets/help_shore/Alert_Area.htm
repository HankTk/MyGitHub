Alert Pane

An alert is a notification generated when a certain condition is met. In SMARTShip, an alert is generated when an input condition or combination of conditions is met by a sensor or an asset.  

Example: You may want to generate an alert when the value from the sensor 'BOOSTER_PUMP_DISCHARGE_PRESSURE_HIGH' is greater than 100.The alert pane appears on the left-hand side of your SMARTShip dashboard. All your pre-configured as well as subscribed alerts appear here.

The colored alert bar in the alert pane categorizes alerts based on their severity. Clicking on the  required color category displays the alerts corresponding to it. By default, only active alerts appears here.

In the above image, clicking on the yellow color displays the alerts belonging to the 'medium' category in the alert pane.

In the alert pane each alert appears in the form of separate cards. Hovering your mouse pointer over an alert card displays the alert details next to it in a separate pane. Please see the below image for reference.

The list view has the list of all alerts in a page format which you navigate. The list view has the following fields:

  • Vessel - Indicates the vessel to which the alert belongs.

  • Alert Name - Name of the alert

  • System - System to which the alert belongs

  • Alert Type - Type of the alert (operational, technical and so on)

  • Alert Description - Description of the alert

  • Active/Inactive - Indicates whether the alert is in active or inactive status. Active alert is indicated by a green circle and inactive alert is indicated using a grey circle

  • Last Updated - Indicates the last updated time of the alert
