TFOC 3 Hourly Report

This report captures the details of the voyage as well as the fuel consumption for each voyage. This reported is automatically generated every three hours for an active voyage (voyage in progress). It computes the fuel consumption from the last received location coordinates at which the report gets generated every 3 hours. This until the voyage is complete.

The report can be accessed from the TFOC app as well. This report data is utilized to plot the Voyage Fuel graph in the TFOC app.

To access Vessel Report:

  1. From the Apps area in Vessel View, select Reports and select TFOC 3 Hourly Reports. The  TFOC 3 Hourly Reports pane appears on the right:

This pane shows reports which have been previously generated. You can also search for the previous instances in a particular time period using Search.

  1. To view an instance of a report, click View under Actions. The report instance appears:

  1. To download the instance of the report in PDF format, click Download under Actions.
