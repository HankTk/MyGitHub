Analyse Alerts

All alerts (alerts that needs to be acknowledged as well the alerts that does not) can be analysed from the alert pop-up box. An alert that does not require acknowledgement is shown in the image below:

To analyse from an alert:

  1. Click Analyse. The analytics  view for the corresponding alert appears:

This area plots a graph based on the values from the alert for analysis. The actions you can do from here is the same you can do from the RMD app. For more information on this, refer to Actions from RMD.

  1. If you have not acknowledged this alert yet, you can enter the required notes and click Acknowledge & Submit.

  1. If you wish to create a case from this alert, click Create Case. For more information on cases and case management, refer to Case Management.
