How to Add Alert Conditions

To add alert conditions:

  1. From the search field, search and select the required sensors. The sensors appear below the arithmetic and logical operators grid.
  2. Click or drag-drop the sensors from here to  the conditions grid as required.
  3. From the Arithmetic and conditional operators grid, click or drag-drop the required operators to the conditions grid to build the condition.
  4. From the numbers grid, click or drag-drop the required numerical value to the conditions grid to build the condition. You can also enter these numerical values directly from your keyboard.
  5. Once the condition expression is completed, click Save. Click Cancel, if you wish to close the Alert Create window without saving.

A sample condition created to generate an alert is given below:

Average (ST_AFT _BEARING_TEMP, ST_FWD_BEARING_TEMP) > 0.1905*ME RPM+ 46.045

Points To Remember While Adding Alert Conditions

  • You can add multiple sensors to the conditions grid to build the expression
  • Click Clear All, to clear all condition elements from the Conditions grid
  • To add any additional elements (sensor, operator or number) before any element in the condition expression, drag-drop the additional element from the corresponding area on top of the expression element in the conditions grid. The condition gets added to the left of the expression element.
  • For detailed information on alert conditions, refer to Add Alert Conditions topic in the SMARTShip help.
