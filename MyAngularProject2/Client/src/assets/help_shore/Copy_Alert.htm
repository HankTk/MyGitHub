Copy Alerts

Allows you to copy vessel specific alerts from one vessel to another. This allows you to re-use the vessel alerts for other vessels rather than creating a new one.

To do this:

  1. Select Copy To from the Alert Management window.

  2. Select the required vessel from the drop-down list. The alerts belonging to the corresponding vessel appears in the pane below:

You can also search and view the required alerts from a vessel using the search field. Click Cancel, if you want to exit the copy function

  1. Select the required vessel to copy the alerts and click Save. The copied alerts are now available on the new vessel.
