Import Alerts        

The Import feature in Alert Management allows you to bulk import alert configurations to your SMARTShip system. This is achieved by importing a tab delimited text file which contains the alert configuration details in a row column format.

A sample configuration file used to import alert configurations is attached below. The custom text file which you use to import the alert configurations must contain the alert details in the same format as the given file below.  To view or edit this file using Microsoft Excel:

  1. Open Microsoft Excel and under File, select Open.

  2. Browse and select the required configuration file. The following Text Import Wizard appears:

  1. Complete all three steps with the default settings. The alert configuration file opens in a tabular format. The headers contains the field names.

To import:

  1. From the Alert Management window, click Import. The Import Alert Configurations box appears:

  1. Click Select Alert Configuration File, navigate and select the required alert configuration custom text file.

  1. Once the import file is selected, click Validate Import File to verify whether the selected file is within the required specifications or format. It also checks whether the selected file has alerts which are already existing in your SMARTShip system in order to avoid any duplication.

  1. If there are no errors, click Create Alert Configuration which appears below the error pane. The alerts are imported to your SMARTShip system.
