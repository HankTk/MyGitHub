List View

The List View tab of the subsystem lists the sensors and its details. It also lists the value of the data incoming from each sensor.

The left pane lists the sensors associated with the asset. Upon selection of a sensor, the corresponding details appear in the right pane.

The following fields appear in the right pane:

  • Sensor Name - Displays the tag names corresponding to the sensor. Each tag is a parameter associated with the sensor.

  • Sensor Description - Displays the details of the tag.

  • Unit - Displays the unit associated with the tag such as meter, degrees and so on.

  • Value - Displays the value of the tag.

The value from the sensors is auto-updated once in every three minutes
