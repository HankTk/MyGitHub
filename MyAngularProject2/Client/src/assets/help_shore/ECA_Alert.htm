ECA Alert

Emission Control Areas (ECAs) are sea areas in which restrictions are established to minimize airborne emissions.  In your SMARTShip system, you can setup alerts before your vessel enters ECA.

To edit the configuration of your ECA alert:

  1. From the Alert Management window, under the Name search field, enter 'ECA'. The ECA Alert appears below:

  1. Click to edit the ECA alert configuration.  The ECA Configuration window appears:

You can setup three alerts before the ship enters ECA from here. Each alert is generated once the ship reaches a particular distance as it approaches the ECA.

  1. Click Edit against an alert to edit. Complete the following fields:

  • Alert Condition - Enter the distance in Nautical Miles (NM) before the ECA for which you wish to generate the alert. Example: You may want to generate an alert when the ship is 200 NMs before the ECA.

  • Alert Description - Enter the description for the alert. Example: '120 NMs before ECA'.

  • Alert Category - Select the alert priority from the list. You can select based on the distance at which the ship is away from the ECA. Example: You may want to select a Critical priority when the ship is 60 NMs away from the ECA.

Similarly, you can setup two more alerts based on the distance.

  1. If you wish to monitor the change over operation of fuel before entering the ECA, select Yes against Monitor Changeover Operation and complete the following details:

    • Alert Condition - Enter the distance before the ECA area for which you want to monitor the changing over of fuel.

    • Alert Description - Enter the required description.

    • Alert Category - Select the required alert priority.

A separate alert is generated before the alert condition provided with the details of the changeover operation.

  1. Select Subscribers, if you want to assign the alert to specific role types. It also allows you to enforce the alert and make it mandatory to acknowledge.

  1. Once all the settings are completed, click Save from the ECA Configuration window.
