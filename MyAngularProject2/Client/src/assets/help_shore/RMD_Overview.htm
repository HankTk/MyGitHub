RMD Overview

Real-time Monitoring and Diagnosis (RMD) allows you to display live data fetched from sensors installed on-board the vessels. The data appears in a graphical representation of raw sensor data. This includes graphical comparison of data from multiple sensors, graphical representation of scaled as well as actual value of sensor data, meaningful analysis and representation of raw sensor data and so on.

To access RMD of a vessel:

  • From the Apps sections of the vessel view, select RMD . The RMD area appears below:

You can also view your favorite charts that you added from the Diagnostics tab of Ship Assets View from here. It appears below CHARTS.
