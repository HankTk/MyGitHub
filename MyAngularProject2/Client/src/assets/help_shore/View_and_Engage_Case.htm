View and Engage Case

Once you have created a case, it gets listed in the Cases pop-up:

The cases appear in a list view in the left pane.

  • Select the required case and the Case Conversation pane on the right side displays the details of the selected case.

Each message that you add to the conversation gets added as a note with a time stamp.

To edit a case:

  1. Click Edit in the top-right corner of the case conversation area. The Edit Case box appears:

  1. Make the required changes and click Save.

To search for a particular case:

  • From the Cases pane in the left, enter the required case name in the Search field.

To filter cases:

  • From the Cases pane in the left, select All Cases. Select either of the following options and the cases with the selected status appears below.

  • To filter the cases in a particular date range, select the required range from the Select Date Range box.
