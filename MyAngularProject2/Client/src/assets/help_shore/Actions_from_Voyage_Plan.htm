Actions from Voyage Plan

Voyage Plans are created from Ship version of the SMARTShip system. However you can view and download the voyage plans created from the shipside.

To access Voyage Plans report:

  • From the Apps area in Vessel View, select Reports and select Voyage Plans from the left pane. The Voyage Plans pane appears on the right:

You can do the following actions from here:

  • View   - Allows you to view the voyage details in a report format. You can also print and download the report from here.

  •  Download - Allows you to download the report in PDF format.

Once the voyage is created, it appears in the TFOC application.
