SMARTShip Dashboard

The SMARTShip dashboard is the home page or the starting point of your SMARTShip system. It provides you with an overview of the vessels that are assigned to you as well as the various actions that you can do using the data available from these vessels.

The SMARTShip dashboard provides you with the following features:

  • Home button to navigate back to the dashboard from any area of the SMARTShip system.

  • Search field to search for the vessels that are assigned to you.

  • Company Card which displays your company details.

  • Vessel Card Area which displays the details of the vessels that are assigned to you. Each vessel appears as separate cards.

  • A Map which displays the real time positions of your vessels in the global map using markers.

  • Alert Area which displays the details of the various alerts generated from the vessels.

No alerts appear in the dashboard, if no vessels are assigned. Also, no alerts are received, if none have been subscribed
