User Management

User Management allows you to create and edit the users of your SMARTShip system. It also allows you map. Once a user is created, you can also do the following actions from the User Management window:

  • Vessel Role Mapping

  • Edit User

  • Reset Password

  • Delete User

To access User Management:

  • Click Settings and select User Management. The User Management window appears:

The following fields appear in the User Management pane:

  • Name - Displays the name of the user.

  • User ID - Displays the user's unique ID.

  • User Type - Displays the type of SMARTShip user. It could be either of the following:

    • Admin

    • Normal

  • Mobile - Displays the mobile number of the user

  • E-mail Address - Displays the E-mail address of the user.

  • Company - Displays your company name.
