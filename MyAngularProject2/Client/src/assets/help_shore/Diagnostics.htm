Diagnostics Tab

This tab allows you to view the data from the tags from each sensor in a diagnostic chart format. The chart or the graph is plotted such that the tag values are in the y-axis and time on the x-axis. You can also group all the sensors and plot the graph as well.

The left most pane lists the asset subsystems associated to the asset. Select the asset subsystem from the list and the graph with the data tag values available from the sensor appears in the graph pane.

The sensor value pane to the extreme right of this window lists the normal sensors and digital sensors separately along with their corresponding values.

The actions you can do from the Diagnostics tab is the same as that of the Diagnostics area of the RMD app. For more information on how to do these actions, refer to Actions from RMD.
