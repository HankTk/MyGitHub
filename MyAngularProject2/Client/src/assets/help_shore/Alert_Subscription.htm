Alert Subscription

The Alert Subscription feature of your SMARTShip system allows you to subscribe the alerts defined and created by your administrator. You can subscribe as well as unsubscribe to alerts from here.

To access Alert Subscription:

  • Click Settings and select Alert Subscription from the menu. The Alert Subscription window appears:

The Alert Subscription window appears empty. You have to select the required category to which the alerts belong in order for them to appear.

The alerts are classified into two categories. They are:

  • Management - Includes all the alerts for all the vessels which are registered under your company. You can further specify the particular system to which the alerts belong.

  • Vessel - Includes alerts coming under a specific vessel. You can further select the particular system to which the alerts belong.

Once you select the required categories, the list of alerts belonging to it appears in the display pane.

The various fields of the alerts subscription display pane are explained below:

  • ALERTS - Displays the alert name.

  • DESCRIPTION - Displays the description of the alert provide by your SMARTShip administrator

  • ACK REQUIRED - Indicates that the alert needs to be acknowledged once it is received.

  • ALERT TYPE - Indicates whether the alert is a technical or operational alert.

Once you determine the alert to subscribe to, you can select any of the following options to subscribe:

  • DASHBOARD - Select, if you want alert to be received as a notification in your SMARTShip dashboard.

  • SMS - Select, if you want the alert to be received as an SMS to the mobile phone number provided in your profile settings.

  • EMAIL - Select, if you want the alert to be received as an E-mail to the E-mail address provided in your profile settings.
