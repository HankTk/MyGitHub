How TFOC works?

The first step in using TFOC is to create the voyage using Voyage Plans from the Ship version of the SMARTShip application.

Voyage Plans can be created or edited only from the ship version of SMARTShip system. You can only view the voyage details from the shore version. If you wish to create or edit a voyage, contact the corresponding vessel master

Once the voyage is created, it gets listed in the TFOC app. The TFOC app then gives the predicted route which is based on the details input by you from Voyage Plans. Before the vessel starts the journey for the selected voyage, it also recommends a forecast route for making the maximum voyage savings as well as different voyage plans such as Best Economy, Best Weather, Fastest  to reach the destination

Once the voyage is complete, the forecast route is removed and the reported or the actual route taken by the vessel appears along with all the associated data such as fuel used, distance travelled and so on. It also provides the actual and predictive fuel consumption for the voyage.

This in turn allows you to understand and analyze the fuel savings you can achieve by taking the routes recommended by TFOC.

Please note that for the correct setting up and functioning of TFOC, you must have access to Voyage Plans and TFOC 3 Hourly Report. Please contact your SMARTShip administrator, if you do not have access to this already.
