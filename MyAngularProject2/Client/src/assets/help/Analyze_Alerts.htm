Analyze Alerts

All alerts (alerts that needs to be acknowledged as well the alerts that does not) can be analyzed from the alert pop-up box. An alert that does not require acknowledgement is shown in the image below:

 

  1. Click Analyze. The analytics  view for the corresponding alert appears:

The following details appear here:

  • Tag Name - Displays the tags from associated sensor data.

To add a custom tag to add to the graph, enter the required tag name in the Search tag name  field and click Add Graph

  • Value - Displays the value of the sensor data

  • Unit - Displays the unit of the sensor data

  • Range -  Displays the range of the sensor data

  • Status - Indicates the status of the alert which depends on the range of the sensor data.  If the sensor value is out of range, then a red colored indicator is used. If the sensor is within the specified range, then a green colored indicator is used.

  1. Click ACK & Close, if you wish to close the analyze area.
