Search Alerts

The search field in the alerts area allows you to search alerts using the title of the category to which the alerts belong:

As an example, in the above image, the 'engine' keyword retrieves alerts belonging to the ENGINE AND MACHINERY category.

The filters in the search field allows you to add further criterion  to your search.

  1. Click and the filter area appears below:

  1. You can apply the following filters from here:

  • Sort By - Allows you sort the alerts using any of the following:

    • Date - Sorts by the date on which the alert was received

    • System - Sorts by the system to which the alerts belongs to

    • Category - Sorts by the criticality category of the alert

  • Date -  Enter the required date range to filter the alerts in that particular date range.

  • System - Select the required system to which the alerts belongs from the list. You can make multiple selections here.

  • Category - Select the criticality category of the alert from the list. You can make multiple selections here.

  1. Once the filter are set, click Apply. The alerts appear as per the selected filters.

Or

If you want to reuse the selected filters again to perform a search, click Save & Apply.
