SMARTShip Dashboard

The SMARTShip dashboard is the home page of your SMARTShip system. It provides with an overview of the vessels that are assigned to you as well as the various actions that you can do using the data available from the vessels.

The SMARTShip dashboard provides you with the following features:

  • Home button to navigate back to the dashboard.

  • Search field to search the vessels that are assigned to you.

  • Company Card which displays your company details.

  • Vessel Card area which displays the details of the vessels that are assigned to you. Each vessel appears as separate cards.

  • Map View which displays the real time positions of your vessels in the global map using markers.

  • Alert Area which displays the details of the various alerts generated from the vessels.

  • Vessel View which provides the detailed description, ability to analyze alerts and view the assets of the selected vessel.

 

No alerts appear in the dashboard, if no vessels are assigned to you. Also, no alerts are received, if you have not subscribed to any alerts.
