Apps in Vessel View

The Apps menu from the vessel view allows you like access various assets, reports and cases for the corresponding vessel. To access Apps:

  1. From top-right corner of the vessel view, click . The Apps menu appear:

  1. The menu items are detailed below:

  • Air Compressor - Provides an overview of the vessel's air compressor.

The reservoir pressure meters shows the values based on the reading from the compressor. The status and condition of the compressor is also indicated here.

  • Cases - Allows you to create and edit cases. For detailed information on this, refer to Case Management Overview.

  • Deck & Engine Report - Allows you to view the deck and engine report. For detailed information on this, refer to Deck & Engine Report.

  • ECA/SECA Report - Allows you to view the ECA/SECA report. For detailed information on this, refer to ECA/SECA Summary Report.

  • Engine Room - Provides an overview of the vessel's engine room.

  • Nav Panel - Provides an overview of the vessel's navigational panel.

  • RMD - Real-time Monitoring and Diagnosis (RMD) allows you to display live data fetched from sensors installed on-board the vessels. For detailed information on this, refer to RMD Overview.

  • SPMS - Ship Performance Monitoring System (SPMS) is an application which monitors the performance of the ship by calibrating fuel consumption during voyages and comparing it with the benchmark (master data). By analyzing data from here, you can find performance related issues and can schedule a proper maintenance of the vessel.

Select the required conditions, start date and end date. The fuel consumption appears in a graphical format.

  •  Vessel Report - Allows you to view the vessel report. For detailed information on this, refer to Vessel Report.
