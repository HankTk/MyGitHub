Alert Card

Each alert appears as cards in the alert area. The following details appear in an alert card:

  • System type of the alert ( like engine and machinery, navigational system or decision support system)

  • Alert name

  • Vessel name

  • Last updated time of the alert

If there are more details in the card, then Show More is available which expands the alert card.

  • To switch back to the normal view, click Show Less.

Alerts that needs to be acknowledged appears with a bold highlighted title. Once acknowledged, the bold highlighting is removed.

The list view has the following fields:

  • Alert Name - Name of the alert

  • System -  System to which the alert belongs

  • Alert Type - Criticality type of the alert

  • Alert Description -  Description of the alert

  • Active/Inactive - Indicates whether the alert is in active or inactive status

  • Last Updated - Indicates the time when the alert was last updated
