Create User

To create a user:

  1. From the User Management window, select Add New. The create new user window appears:

  1. Complete the following details:

  • Name - Enter the required name of the user.

  • User ID - Enter the required user ID for the user. This ID is used to login into the SMARTShip system.

  • Phone No - Enter the user's phone number along with the country code.

  • Email - Enter the E-mail address of the user.

  • Usertype - Select the required user type. You can select either of the following:

    • Admin - If selected, the user has administrator privileges to your SMARTShip system which includes creating user, reports, alerts and so on.

    • Normal - If selected, the user has access to the regular features of your SMARTShip system and does not have any administrator privileges.

  1. Once completed, click Save. The user is created and appears in the User Management pane and an E-mail is sent to the user's E-mail address with the username and password.
