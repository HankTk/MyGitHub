Overview

SMARTShip as A DIGITAL ENTERPRISE is an Industrial Internet of Things (IIoT) based system capable of serving multiple number of vessels of your company. It connects all the components or equipment on-board the vessel with each other as well as to the Internet. This makes the data on every single component available to the SMARTShip user which in turn allows remote monitoring, diagnostics and control, decision support systems, real time analytics, maintenance and so on.

How it Works?

The SMARTShip system collects data from the various components of your vessel using data acquisition units such as sensors installed on-board the vessel. This acquired data is used by the SMARTShip application which acts as a central console to locally or remotely monitor and manage the connected devices.

Features and Benefits

The SMARTShip platform provides the following features:

  • Interconnected devices and end-to-end security

  • Remote monitoring, diagnostics and control

  • Data analytics and real-time analytics

  • Decision support system using Alert Management System

  • Create cases based on alerts or on demand using Case Management System

  • Descriptive, predictive and prescriptive maintenance

  • Live CCTV feed from the vessel for remote monitoring

Using the above capabilities of SMARTShip, you are able to achieve the following benefits:

  • Cost optimization by real-time monitoring of devices and on-board machinery using a centralized system

  • Increase the overall operational efficiency by proactive and corrective actions using real time data

  • Data security
