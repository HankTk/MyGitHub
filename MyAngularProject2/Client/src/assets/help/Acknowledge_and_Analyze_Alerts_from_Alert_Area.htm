Acknowledge an Alert

An alert that requires to be acknowledged is highlighted in bold.

  1. Click the required alert from the alert area. The alert appears as a pop-up in the map area.

  1. Click Acknowledge. The alert is acknowledged and the Close button appears.

Note If you want to add notes and then acknowledge the alert, select Acknowledge & Submit

The number next to the Close button (0/2, 0/1, 1/1 and so on) indicates the following:

'number of users acknowledged/ the number of users who must acknowledge the alert'

Example: 0/2 against the Close button indicates that zero users have acknowledged the alert/2 users must acknowledge it. Similarly, 1/1 indicates 1 user has acknowledged/1 user must acknowledge the alert.

  1. Click the number and the following pop-up appears:

The below details appear in this pop-up:

  • Number of users who acknowledged the alert

  • Names of the users who acknowledged

  • Date when the alert was acknowledged

Once acknowledged, you can choose to close the alert or proceed to analyzing it.

Note You can also create a case based on alert. To do this, click Create Case from the Analyze box. For detailed information on this, refer to Create and Edit Case
