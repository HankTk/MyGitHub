Vessel Role Mapping

Vessel Role Mapping allows you to map a user to the various role types under a specific vessel created using Role Management. You can map multiple vessels and the various roles which come under them.

Once a user is mapped to a role, the user has access to the screens selected in the Role.

To map a user to a specific role:

  1. From the User Management pane, select against the required user. The Assign Vessel Role window appears:

  1. Select the required vessel from the Vessel list. The roles corresponding to the vessel appears under the Role list.

  1. Select the required roles from the list. To further add roles from another vessel, click Add Row. You can select other vessels and map the roles corresponding to the vessel.

  1. Once the row mapping is completed, click Save. The mapping is completed.
