Add Alert Conditions

An alert is generated when a condition set by you is met by the sensor or an asset.  

Example: You may want to generate an alert when the value from the sensor 'BOOSTER_PUMP_DISCHARGE_PRESSURE_HIGH' is greater than 100.

You can also combine multiple conditions and values using various operators and numerical values.

To set the conditions:

  1. Once the alert details are completed from the left pane, hover over to the right pane of the alert create window.

  1. Build your own custom conditions using the following:

  • Search - Allows you to search and select the various sensors to add to the alert condition. Once a sensor is selected, it appears in the grid below the operators. See the below image for reference.

To add this to the condition, click or drag-drop  the sensor to add to the conditions grid below.

Similar to above, you can add multiple sensors to the conditions grid.

  • Arithmetic and conditional operators grid - This grid allows you to add various arithmetic operators like +, -, * and so on and conditional operators like average, sum, previous to the condition. You can click or drag-drop to add them to the conditions grid.

  • Numbers - The numbers grid allows you to add numerical values to your condition. You can click or drag-drop to add them to the conditions grid.

Click Clear All, if you want to clear the elements from the conditions grid

  1. Once the condition is built, click Save. The alert is created.
