View and Engage Case

Once you have created a case, the cases appear in the Cases area under Apps in the vessel view:

The cases appear in a list view in the left pane. The color code against the case indicate the alert severity.

Select the required case and the Case Conversation pane on the right side displays the details of the selected case.

Each message that you add to the conversation gets added as a note with the time stamp.

To edit a case:

  1. Click Edit in the top-right corner of the case conversation area. The Edit Case box appears:

  1. Make the required changes and click Save.

To search for a particular case:

  • From the Cases pane in the left, enter the required case name in the Search field.

To filter cases:

  • From the Cases pane in the left, select All Cases. Select either of the following options and the cases with selected status appears below.
