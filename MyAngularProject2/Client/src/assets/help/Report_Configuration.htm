Report Configuration

Report configuration List allows you to create new report configurations based on several pre-defined templates. You can also view the list of report configurations created as well as edit or delete them  from here.

To access Report Configuration list:

  • Click Settings . and select Report Configuration from the menu. The Report Configuration List window appears:

The following fields appear in the reports pane:

  • Report Name - Displays the report name.

  • Vessel Name - Displays the vessel name corresponding to the report.

  • Frequency - Displays the rate (daily or weekly) at which the report is generated.

  • Report Timing - Displays the day and time at which the report is generated based on the frequency.

  • Output Format - Displays the format at which the report is generated. The format can be PDF or HTML.

  • Last Generated - Displays the last generated time of the report.

You can view the reports specific to a vessel. To do this:

  • Select the required vessel from the Select Vessel list.

Or

Search the required vessel in the Search field.
