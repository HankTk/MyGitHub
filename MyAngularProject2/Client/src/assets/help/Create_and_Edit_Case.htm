Create and Edit Case

There are two methods in which you can create a case:

  • Create Case from an alert

  • Create Case from Apps in vessel view

Create Case from Alert

To create a case from an alert:

  1. Select the required alert from the alert area. The alert box appears in the map area:

  1. Click Create Case. The Create Case box appears:

  1. Complete the following details:

  • Case Title - Enter a name for the case.

  • Case Description - Enter a brief description.

  • Category - Select the required category from the list.

  • Severity - Select the case severity from the drop-down list.

  1. Once the details are completed, click Save. The below message appears and the case is created.

Create Case from Apps in Vessel View

To create case for a vessel from its vessel view:

  1. Determine the required vessel and select the vessel from either of the methods mentioned in Accessing Vessel View.

  2. Click Apps and select Cases. The Cases area for the corresponding vessel appears:

  1. Click . The Create Case box appears:

  1. Complete the fields as per the instructions given to create a case from an alert and click Save.
