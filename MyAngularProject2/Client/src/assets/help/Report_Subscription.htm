Report Subscription

This feature allows you to subscribe the various pre-defined reports set by your administrator. The Report Subscription list displays the list of reports and the ability to subscribe and unsubscribe the reports from here.

To access Report Subscription:

  • Click Settings and select Report Subscription. The Report Subscription List appears:

The various fields of your Report Subscription list is described below:

  • REPORT TITLE - Indicates the title of the pre-defined report. The following are the three pre-defined reports in your SMARTShip system:

    • Deck and Engine E-Log Book Report

    • Vessel Report

    • ECA/SECA Summary Report

For detailed information on Reports, refer here

  • VESSEL NAME - Indicates the vessel on which the report is based.

  • FREQUENCY - Indicates the time period in which the report is received. The reports are received in a daily or weekly basis.

Your administrator can also specify the day and time for a weekly schedule and time for a daily schedule of the reports

  • SUBSCRIBE - Select this, if you want to subscribe the corresponding report. Unselect, if you want to unsubscribe.

Once subscribed, the report is send in an E-mail in the format set by the administrator, once it is generated. The formats available are .PDF or .HTML.
