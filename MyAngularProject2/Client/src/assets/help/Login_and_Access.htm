Accessing SMARTShip

To access SMARTShip:

  1. Open your web browser and in the address bar, enter the following URL:

https://smartship-test.run.aws-jp01-pr.ice.predix.io

The SMARTShip login window appears:

 

The preferred web browser for using SMARTShip application is Google Chrome or Mozilla Firefox

  1. Complete the following the details:

  • User Id - Enter your SmartShip user ID

  • Password - Enter your SmartShip password

  1. Click LOGIN. The SMARTShip dashboard appears:
