Settings

The Settings feature  allows you to manage and subscribe the various alerts and reports  in your SMARTShip system.  It also allows you manage and create roles and users in your account. To access Settings:

  1. From the top-right corner of your SMARTShip dashboard, click Settings . The Settings menu appears:

The Settings menu contains the following menu items:

  • Alert Management - Allows you to create and manage alerts.

  • Alert Subscription - Allows  you to subscribe alerts.

  • Report Configuration - Allows you to create and define reports based on pre-defined templates.

  • Report Subscription - Allows you to subscribe reports.

  • User Management - Allows you to create users based on the user type, manage users and map users to vessels

  • Role Management - Allows you to create and manage roles with access to specific screens of SMARTShip system.
